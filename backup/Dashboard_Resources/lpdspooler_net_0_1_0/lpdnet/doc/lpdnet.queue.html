lpdnet.queue Namespace

Namespace hierarchy

Classes

Class Description
PrintQueue The PrintQueue is just a queue that only accepts print jobs.
Queue Queue of objects
QueuedObject Encapsulates any Object that is stored in a Queue
QueuedPrintJob A copy of a printJob plus the jobId used by the PrintQueue to store the opriginal copy of this PrintJob.
QueuedPrintJobInfo Encapsulates the PrintJob that is stored in the Queue. A copy of the actual stored PrintJob is returned. This class makes it easy to expose more attibutes of the Print Job as necessary.
QueueMonitor Monitors a single print queue and process all queued jobs using the queue job handler. A new QueueMonitor should be defined for each PrintQueue.
Queues This class maintains all of the Print queues and contains all of the logic for performing operations on them.
